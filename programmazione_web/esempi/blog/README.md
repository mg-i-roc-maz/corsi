# Blog Web - Istruzioni di avvio

## Requisiti

- Node.js installato
- Nella cartella `blog` deve essere presente il file `index.html` e `server.js`

## Installazione dipendenze

Apri il terminale nella cartella `blog` ed esegui:

```sh
npm install express
```

## Avvio del server

Esegui:

```sh
node server.js
```

La pagina sarà disponibile su:

```
http://localhost:8081
```

## API post

Assicurati che il backend API (Java/Spring) sia avviato su:

```
http://localhost:8080/api/posts
```

## Funzionalità

- Visualizza i post del blog
- Visualizza il dettaglio di ogni post
- Crea nuovi post
